Most SEO tools leave you in a dashboard. RankEarly Skills let your agent research, draft, assess, and polish content — you maintain the SEO memory, choose the angle, and review the result.

Works with Claude, Codex, OpenClaw, and any MCP-compatible agent.

## Available skills

| Skill | What it does |
|-------|-------------|
| [Topic Research](https://www.rankearly.pro/skills/topic-research) | Find what to write by expanding a keyword library and organizing it into themes and keyword clusters |
| [SERP Gap Analysis](https://www.rankearly.pro/skills/serp-gap-analysis) | Validate whether a keyword or topic is worth writing before you commit |
| [Content Researcher](https://www.rankearly.pro/skills/content-researcher) | Gather what current ranking content covers and spot gaps your content can fill |
| [Blog Title Generator](https://www.rankearly.pro/skills/blog-title-generator) | Turn an approved angle into strong title options for search and click-through |
| [Blog Creator](https://www.rankearly.pro/skills/blog-creator) | Create SEO blog posts — research, outline, and full writing |
| [SEO Memory](https://www.rankearly.pro/skills/seo-memory) | Keep product context, positioning, and customer language consistent across every workflow |

## Install

Install all skills:

```bash
npx skills add rankearly/rankearly-skills
```

Or install a single skill:

```bash
npx skills add rankearly/rankearly-skills --skill serp-gap-analysis
```

### Manual install

Download and extract the `SKILL.md` into your project's `.claude/skills/` directory.
